# UE4Fighter-Showcase
My first UE4 project, a singleplayer first-person fighter. Special focus was placed on gamefeel and the user experience. 

## Highlights
- Fully-formed player moveset, including complex movement options and 2 unique weapons
- Three enemy types, each with dynamic attack patterns and behaviors
- Fully implemented user settings, including graphics, controls, and gameplay options
- Fully implemented audio feedback
- Two cel-shaded environments

## Technical Focus
- Unreal Engine 4.25
- C++ syntax and design principles
- 3D modeling and animation
- Vector and matrix math
- Satisfying audio-visual feedback
